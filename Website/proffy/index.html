Proffy

A native code profiler for Windows

About

Free software for profiling native code applications under Windows.

  • Sampling only, no intrusive instrumentation.
  • Call graph profiling.
  • GUI-less. Include a single C++ header file into your application to control exactly when the profiler starts and stops.
  • Results saved to XML and viewable in your browser.

Created by Paul Richards and released as free software under the GPLv3.
